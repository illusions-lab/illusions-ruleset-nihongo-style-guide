# nsg-ellipsis-style — 省略を示すリーダー

- **ruleId**: `nsg-ellipsis-style`
- **ガイドライン**: 日本語スタイルガイド (`jtca-style-guide-3`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 (`novel`) / 公用文 (`official`) / ブログ (`blog`) / 学術 (`academic`) / SNS (`sns`)

## 意図

文の途中の省略を示すリーダーは三点リーダー（…）を用いる。リーダーの代用として半角ピリオドを連続させた箇所（...）を検出する。

## 出典

日本語スタイルガイド 第3版（テクニカルコミュニケーター協会）第1章 1.3.3節

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `続きは…次回へ。` | 指摘なし |
| 誤例 | `続きは...次回へ。` | `…` を提案 |

## 実装メモ

- 半角ピリオド2個以上の連続に限定。単独の小数点・項番には反応しない。相対パス `../` は lookahead で除外。
