# nsg-katakana-kanyo — カタカナ表記の慣用形（ヴ音）

- **ruleId**: `nsg-katakana-kanyo`
- **ガイドライン**: 日本語スタイルガイド (`jtca-style-guide-3`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

原語が「V」音のカタカナ表記は、慣用として「バ・ビ・ブ・ベ・ボ」を充てる。ヴァ・ヴィ・ヴ・ヴェ・ヴォ・ヴュを検出し、対応する慣用形を提案する。

## 出典

日本語スタイルガイド 第3版（テクニカルコミュニケーター協会）第1章 1.3.2節

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `バイオリンを弾く。` | 指摘なし |
| 誤例 | `ヴァイオリンを弾く。` | `バ` を提案 |

## 実装メモ

- `ctx.toolkit.regexReplace` で `/ヴ[ァィェォュ]?/` を検出し、対応表で慣用形へ変換。
- 固有名詞でヴ表記を意図する場合があるため severity=warning（提案）に留める。
