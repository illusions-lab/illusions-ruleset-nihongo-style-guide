# nsg-koto-formalnoun — 形式名詞「こと」の漢字表記

- **ruleId**: `nsg-koto-formalnoun`
- **ガイドライン**: 日本語スタイルガイド (`jtca-style-guide-3`)
- **レベル**: L2（形態素解析）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

日本語スタイルガイド 第3版 第1章「用字・用語」§1.2.2「品詞や意味による漢字とひらがなの使い分け」では、同じ言葉でも品詞・意味によって漢字とひらがなを書き分けることを求めている。「こと（事）」については、具体的な事物を表す普通名詞の用法では漢字「事」、形式名詞の用法ではひらがな「こと」を充てる例が示されている。

形式名詞（補助名詞）として用いる「事」をひらがな「こと」に統一することを提案する。

## 出典

日本語スタイルガイド 第3版（テクニカルコミュニケーター協会）第1章 1.2.2節

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `変更することもある。` | 指摘なし |
| 誤例 | `変更する事もある。` | 形式名詞「事」を「こと」に変更する提案 |

## 実装メモ

- kuromoji による形態素解析で `pos="名詞"` かつ `pos_detail_1="非自立"` のトークン「事」のみを対象とする。
- 普通名詞として用いる「事」（`pos_detail_1="一般"` 等）は対象外。
- 固有名詞・サ変接続は対象外。
- `lint()` は L2 ルールのため空配列を返す（morphological エンジン用）。
