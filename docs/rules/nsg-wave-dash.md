# nsg-wave-dash — 範囲を示す波記号

- **ruleId**: `nsg-wave-dash`
- **ガイドライン**: 日本語スタイルガイド (`jtca-style-guide-3`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

範囲を示す記号は波記号（〜）に統一する。数値範囲を半角チルダ（~）で表している箇所を検出する。

## 出典

日本語スタイルガイド 第3版（テクニカルコミュニケーター協会）第1章 1.3.3節

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `10〜20ページを読む。` | 指摘なし |
| 誤例 | `10~20ページを読む。` | `〜` を提案 |

## 実装メモ

- 「数字 ~ 数字」の形に限定し、URL やパス先頭の ~ には反応しない。
