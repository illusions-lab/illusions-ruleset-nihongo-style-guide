# nsg-unit-time — 速度の単位表記

- **ruleId**: `nsg-unit-time`
- **ガイドライン**: 日本語スタイルガイド (`jtca-style-guide-3`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

単位の表記には原則として国際単位系（SI）の英字記号を使う。速度は「km/時」ではなく「km/h」と表記する。

## 出典

日本語スタイルガイド 第3版（テクニカルコミュニケーター協会）第1章 1.3.3節

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `最高120km/hで走る。` | 指摘なし |
| 誤例 | `最高120km/時で走る。` | `km/h` を提案 |

## 実装メモ

- 「距離単位＋/時」「km毎時」の形に限定し、速度文脈以外の「時」には反応しない。
