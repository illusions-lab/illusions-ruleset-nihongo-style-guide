# nsg-katakana-long-vowel — カタカナ長音符号の省略

- **ruleId**: `nsg-katakana-long-vowel`
- **ガイドライン**: 日本語スタイルガイド (`jtca-style-guide-3`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

一般的な外来語の語末長音符号（ー）は省略しない。語末の長音が省略された代表的な一般用語を検出し、長音符号付きの表記を提案する。

## 出典

日本語スタイルガイド 第3版（テクニカルコミュニケーター協会）第1章 1.3.2節

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `新しいコンピューターを買う。` | 指摘なし |
| 誤例 | `新しいコンピュータを買う。` | `コンピューター` を提案 |

## 実装メモ

- `ctx.toolkit.regexReplace` で「省略形＋直後が長音でない（lookahead）」を検出。
- 収録は省略が一般化しやすい代表的な一般用語に限定（技術用語は方針が分かれるため対象外）。
