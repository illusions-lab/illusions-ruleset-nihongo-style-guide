# nsg-fullwidth-alnum — 英数字の半角統一

- **ruleId**: `nsg-fullwidth-alnum`
- **ガイドライン**: 日本語スタイルガイド (`jtca-style-guide-3`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

和文中の英字・数字は半角に統一する。全角英字（Ａ-Ｚ, ａ-ｚ）・全角数字（０-９）を検出し、半角への置換を提案する。

## 出典

日本語スタイルガイド 第3版（テクニカルコミュニケーター協会）第1章 1.3.3節

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `バージョン3を使う。` | 指摘なし |
| 誤例 | `バージョン３を使う。` | `3` を提案 |

## 実装メモ

- 連続する全角英数字を1スパンにまとめ、NFKC 正規化で半角へ変換。全角記号は対象外。
