# nsg-unit-case — 単位記号の大文字小文字

- **ruleId**: `nsg-unit-case`
- **ガイドライン**: 日本語スタイルガイド (`jtca-style-guide-3`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

SI単位記号には大文字・小文字の使い方が定められている（mg・kg は小文字、W・V・A は大文字、Hz・Pa・kW は大小混在）。数値に続く単位記号の大小誤りを検出する。

## 出典

日本語スタイルガイド 第3版（テクニカルコミュニケーター協会）第1章 1.3.3節

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `重さは5kgです。` | 指摘なし |
| 誤例 | `重さは5Kgです。` | `kg` を提案 |

## 実装メモ

- 「半角数字＋単位記号」に限定し、直後に英字が続く語（型番等）は lookahead で除外。
