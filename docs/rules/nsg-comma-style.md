# nsg-comma-style — 読点の統一（コンマ）

- **ruleId**: `nsg-comma-style`
- **ガイドライン**: 日本語スタイルガイド (`jtca-style-guide-3`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 (`novel`) / 公用文 (`official`) / ブログ (`blog`) / 学術 (`academic`) / SNS (`sns`)

## 意図

実用文の読点は点（、）に統一する。和文の途中に全角コンマ（，）が読点として使われている箇所を検出する。

## 出典

日本語スタイルガイド 第3版（テクニカルコミュニケーター協会）第2章 2.8.1節

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `保存し、終了する。` | 指摘なし |
| 誤例 | `保存し，終了する。` | `、` を提案 |

## 実装メモ

- 直前が日本語文字のときに限定し、数字の位取り（1，000）には反応しない。
- span は「，」1文字のみを置換対象にする。
