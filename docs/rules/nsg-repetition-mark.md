# nsg-repetition-mark — 繰り返し符号（々）の誤用

- **ruleId**: `nsg-repetition-mark`
- **ガイドライン**: 日本語スタイルガイド (`jtca-style-guide-3`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 公用文 (`official`) / 学術 (`academic`) / ブログ (`blog`)

## 意図

繰り返し符号「々」は同一漢字の繰り返し（国々・山々）を示すが、複合語境界（研究会会長・物理理論）には使わない。複合語境界での「々」誤用を検出する。

## 出典

日本語スタイルガイド 第3版（テクニカルコミュニケーター協会）第1章 1.3.3節

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `研究会会長に就任する。` | 指摘なし |
| 誤例 | `研究会々長に就任する。` | `研究会会長` を提案 |

## 実装メモ

- 複合語境界の誤用に当たる語形を curated list として明示し、その語に限って検出。正しい畳語（国々・人々）には反応しない。
