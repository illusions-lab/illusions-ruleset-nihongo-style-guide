# nsg-period-style — 句点の統一（ピリオド）

- **ruleId**: `nsg-period-style`
- **ガイドライン**: 日本語スタイルガイド (`jtca-style-guide-3`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 (`novel`) / 公用文 (`official`) / ブログ (`blog`) / 学術 (`academic`) / SNS (`sns`)

## 意図

実用文の句点は丸（。）に統一する。和文の文末に全角ピリオド（．）が句点として使われている箇所を検出する。

## 出典

日本語スタイルガイド 第3版（テクニカルコミュニケーター協会）第2章 2.8.1節

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `これは新製品です。` | 指摘なし |
| 誤例 | `これは新製品です．` | `。` を提案 |

## 実装メモ

- 直前が日本語文字（ひらがな・カタカナ・漢字・長音）のときに限定し、項番（2．3）や小数点には反応しない。
- span は「．」1文字のみを置換対象にする。
